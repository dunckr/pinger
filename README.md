# pinger

Schedule requests to endpoints for uptime. For more information about cron patterns see [here](http://crontab.org/).

#### Install

`npm i`

#### Run

`npm run watch`

#### Env vars

* ENDPOINT="http://www.google.com"
* CRON_PATTERN="* * * * * *"

### Test

`npm run test`

### Deploy

`git push heroku master`
